开发工具及使用

+++

1. listary

  • 双击Ctrl全局搜索
  • 在文件管理器中双击空白处进行列表呼出

2.cmd

  • 在当前文件目录的显示栏下输入 cmd ,即可在当前路径下打开命令行

3,stb_image简单图像库

  • 重点关注三个头文件

4.Windows OS

  • Shift+F10:鼠标右键快捷键
  • Win+Ctrl+←/→：窗口切换，保命技巧

5.RenderDoc

  • 可集成到VS或者Unity3D中

6. Typora

  • Ctrl+/：源代码模式进入或者退出

7. Visual Studio Code

  • Ctrl+Shift+P:调出主命令框
  • Ctrl +K选中后，再按V：预览markdown效果

8. Scons

  • 基于python的软件构建工具

9.Python

在同一计算机上同时配置两个不同版本的Python时，例如Python2.7以及Python3.6两个版本，可以在Python27文件夹下将python.exe文件更名为python27.exe，然后在用户环境变量下的Path下维护两个版本的Python地址，每个版本两个地址（一个Python文件夹地址，一个Script地址）
